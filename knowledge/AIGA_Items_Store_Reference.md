## STORE — EMPIRE COIN PRICES

The Items Store uses Empire Coins as currency and has a fixed (non-refreshing) inventory.

### Skill Scrolls

| Item | Empire Coin Cost |
|---|---|
| Legendary Skill Scroll | 1,000 |
| Epic Skill Scroll | 175 |
| Rare Skill Scroll | 50 |
| Name Change Scroll | 500 |

**AIGA advisory:** Legendary Skill Scrolls are needed to upgrade legendary customisable skills. At 1,000 EC each, saving them from VIP daily rewards (VIP 12+) is far more efficient than purchasing. Only buy from store if a skill upgrade is critically needed before a major event.

---

### Skill Unlock Medals — Legendary Tier (8,000 EC each)

These unlock Legendary customisable skills. All cost 8,000 Empire Coins per medal.

Fearless Retribution, Forceful Retribution, Strategy Master's Gift, Flash of Inspiration, Furious Charge, Critical Insight, Whirlwind Sweep, Triple Offensive, King's Blade, Weak Spot Attack, Crushing Impact, Darkness Strike, Ultimate Strategy, Immortal Army, Bloodthirst, Peerless Strike, Tactical Pursuit, Conflagration, Earth Crush, War Tactic, Righteous Judgement, Peaceful Haven, Tenacity, Savage, Spirited Pursuit, Act of Mercy, Roar of Victory, Fatal Blow, Berserker, Double Attack, Warrior's Hymn, Protracted Battle, High Spirit, Coordination, Siege and Plunder, Fearless Frontrunner, Siege Breaker, Windfall, Supplies Transportation, Organized Retreat, Battle Roar, Ripper Tiger, Executor, Pay in Blood, Dark Flag, War Elephant, Entrenchment

**AIGA advisory:** At 8,000 EC per Legendary skill medal, this is a significant investment. To unlock a Legendary skill requires the medal + reaching the required hero level. Compare: 8,000 EC = 1.3 × 24h universal speedups. Prioritise which Legendary skills are essential (Flash of Inspiration for Sun Tzu/Suleiman, Whirlwind Sweep for pike leads, etc.) before purchasing.

### Speedups (Empire Coins)

All speedup types (Universal, Building, Forging, Healing, Research, Training) follow identical pricing:

| Duration | EC Cost | EC per minute |
|---|---|---|
| 1m | 5 | 5.00 |
| 5m | 21 | 4.20 |
| 15m | 63 | 4.20 |
| 60m | 250 | 4.17 |
| 3h | 750 | 4.17 |
| 8h | 2,000 | 4.17 |
| 15h | 3,750 | 4.17 |
| 24h | 6,000 | 4.17 |
| 3d | 18,000 | 4.17 |
| 7d (Forging only) | 42,000 | 4.17 |
| 15d (Forging only) | 90,000 | 4.17 |
| 30d (Forging only) | 180,000 | 4.17 |

**Key reference values:**
- 1 day speedup = 6,000 EC
- 1 day speedup = 1,440 minutes × 4.17 EC/min
- Legendary skill medal (8,000 EC) = ~1.3 day speedups
- Legendary Advent single spin (900 EC) = ~3.6h of speedup equivalent

**AIGA advisory:** Speedups are the highest-value use of Empire Coins in most situations. The Legendary Advent wheel at 900 EC/spin gets you 1,000 MGE pts (Day III) or hero medals — generally better value than buying raw speedups unless you have a critical time-sensitive build/research to complete.

## STORE CATEGORIES — WHERE TO SPEND WHAT

| Store | Currency | Refresh | Best items to buy |
|---|---|---|---|
| Items Store | Empire Coins | Fixed (always available) | Skill scrolls, skill medals, speedups |
| Merits Store | Merits | Daily | Food, Wood, Stone, Training speedups, XP Tomes |
| Military Exercises | Exercise tokens | Weekly | Skill scrolls, food/resources, speedups |
| Arena Store | Arena Silver | Weekly | XP Tomes, Skill Points, Hero Medals, Recruitment Tokens |
| Island Tactics | Tactic Coins | Weekly | XP Tomes, Skill Points, Blueprints, Crystals, Forging speedups, Iron Meteorite, Forging Tools |
| Alliance Treasury | Alliance Coins | Weekly | Resource Chests, Building/Research speedups, Skill Points, Recruitment Tokens |
| Store (General) | Store tokens | Weekly | Hero Medals, Recruitment Tokens, XP Tomes, Building/Research/Forging speedups, Iron Meteorite, Forging Tools |

**AIGA priority by store:**
- **Island Tactics** — buy every gear material item (Iron Meteorite, Forging Tools, Crystals) every reset. Non-negotiable.
- **Arena Store** — prioritise Skill Points and hero medals for current march heroes.
- **Alliance Treasury** — Building/Research speedups and Skill Points.
- **Merits Store** — Training speedups and XP Tomes daily.

## REDEEM CODES

AoEM occasionally releases free gift codes via official social media (Discord, Facebook, Twitter). Codes provide speedups, Empire Coins, resources, and similar items.

**How to redeem:**
1. Tap profile icon (top left)
2. Tap Settings (lower left)
3. Tap the gift chest icon labelled "Gift Redemption"
4. Enter code and tap Redeem

**Rules:** Each code is usable once per account. Codes are time-limited and expire. All codes are created by official developers only — they are safe to use and will not risk your account.

**Where to find current codes:** Official AoEM Discord, Facebook, Twitter/X. Community sites like aoemobileguides.com also aggregate them. *AIGA does not maintain a live code list as codes expire — check official channels for current valid codes.*
