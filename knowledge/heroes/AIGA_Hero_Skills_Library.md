## Overview

This document catalogues all known learnable hero skills in Age of Empires Mobile. Each skill entry includes rarity, function type, key mechanical values, and eligible heroes.

**Rarity tiers:** Elite < Epic < Legendary

Skill damage rates are base values at max skill level. All values sourced directly from in-game data as provided.

## Key Advisory Notes

### Top damage skills by rate
1. Roar of Victory — 107.28% (Legendary, after normal attack, + rage)
2. Spirited Pursuit — 86.86% (Legendary, after normal attack)
3. Battle Roar — 89.40% (Legendary)
4. Boulder Trap — 76.93% (Epic, charge)
5. Weak Spot Attack — 74.74% (Legendary, after normal attack, + damage buff)

### Never equip gathering skills on combat march heroes
Efficient Harvest, Windfall, Rapid Gathering, Load Boost, etc. are zero value in combat. Flag immediately if found on M1-M4 heroes.

### Healing anti-synergy: Darkness Strike
Darkness Strike reduces enemy healing by 10.10%. Pair on a hero facing heavy recovery opponents. Avoid wasting on opponents without healing builds.

### Best skills by role
- **M1 lead (damage):** Roar of Victory / Spirited Pursuit / Weak Spot Attack + Act of Mercy
- **M1 support (debuff):** Armor Piercer / Critical Insight / Crushing Impact
- **M2 support (recovery):** Tenacity / Peaceful Haven / War Tactic
- **Rally lead:** Furious Charge stack + Whirlwind Sweep stack for sustained output
- **Gathering M5:** Windfall + Supplies Transportation (Diao Chan exclusive combo — highest load + resource bonus)
